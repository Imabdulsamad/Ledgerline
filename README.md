Ledgerline 📊
Ledgerline is a modern, high-performance personal finance tracker designed for clarity, efficiency, and privacy. By focusing on a frictionless user experience, Ledgerline empowers you to manage expenses, monitor budgets, and visualize spending trends without the bloat of traditional financial software.

🚀 Features
Transaction Management: Seamlessly add, edit, and remove transactions.

Budget Oversight: Define budget goals and keep track of upcoming bills.

Insightful Analytics: Visualize financial health through dynamic charts and intuitive category breakdowns.

OCR Receipt Handling: Streamline entry with OCR-enabled drag-and-drop receipt processing.

Local-First Privacy: Data is persisted locally, ensuring your financial information stays on your machine for maximum security and performance.

Dynamic UI: Toggle between light and dark modes to suit your workflow.

🛠 Tech Stack
Ledgerline is built with a focus on maintainability, speed, and modern developer patterns:

Framework: React

Build Tool: Vite

Styling: Tailwind CSS

State & Logic: Custom React hooks for efficient state and theme management.

Utilities: Custom modules for robust CSV handling, financial formatting, and unique ID generation.

📦 Getting Started
Prerequisites
Node.js (LTS version recommended)

npm or yarn

Installation
Clone the repository:

Bash
git clone https://github.com/your-username/ledgerline.git
cd ledgerline
Install dependencies:

Bash
npm install
Run the development server:

Bash
npm run dev
Open http://localhost:5173 in your browser.

💡 Why Ledgerline?
In an era of over-engineered financial platforms, Ledgerline stays true to the "less is more" philosophy. We provide a clutter-free dashboard that allows you to focus on what matters: your financial health. By keeping your data local, we guarantee faster load times and the peace of mind that your sensitive data remains yours.
